i. ¿Qué aprendiste sobre el flujo de trabajo con Git y GitHub?
Aprendí a crear repositorios, clonarlos, hacer cambios con git add, git commit y git push. También aprendí a trabajar con ramas y fusionarlas con merge.

ii. ¿Cómo fue la experiencia de trabajar en pareja? ¿Encontraste algún conflicto? ¿Cómo lo resolviste?
Realicé el laboratorio de manera individual, por lo tanto no tuve conflictos al trabajar.

iii. ¿Qué ventajas crees que tiene usar Git en proyectos de programación?
Permite llevar un control de versiones, trabajar con ramas sin afectar el proyecto principal, colaborar con otras personas y mantener un historial de cambios.
